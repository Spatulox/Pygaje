VARIABLES :
- Scope
- Type : string / int / array

TABLEAU :
- Initialisation
- Accès
- Remplacement

CLASSES :
- Variables
- Méthode
- Classes enfant qui héritent des méthodes et des variables des parents
- Gestions des Erreurs :
    - Nombres de paramètres
    - Méthode/var qui n'existe pas

FONCTIONS :
- Fonction void/return
- Fonctions avec/sans paramètres
- Fonctions récursives (gérée avec un while)
- Passage par référence (c'est une copie dans le scope suivant (pour "lier" la nouvelle variable à la var parent), puis remplacement de la valeur originale par la copie à la fin de la fonction)
- Gestion des Erreurs
    - Pas de break dans une fonction (sans être dans une boucle)
    - Nombre de paramètres

FONCTIONS PAR DEFAUT :
- print
- len
- debug
